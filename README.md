# 🎙️ Voice Assistant

A Python-based Voice Assistant that listens to user voice commands and performs various tasks such as opening websites, searching information, telling the time, and responding through speech.

## 🚀 Features

- Voice command recognition
- Text-to-speech responses
- Open websites and applications
- Search information on the internet
- Tell current date and time
- User-friendly and interactive

## 🛠️ Technologies Used

- Python 3
- SpeechRecognition
- pyttsx3
- Webbrowser
- Datetime

## 📂 Project Structure

```text
Voice-Assistant/
│
├── voice_assistant.py
├── requirements.txt
└── README.md
```

## ▶️ How to Run

1. Install Python 3.
2. Install required libraries:

```bash
pip install SpeechRecognition pyttsx3
```

3. Run the program:

```bash
python voice_assistant.py
```

## 💡 Example Commands

```text
"Open Google"
"What is the time?"
"Search Python programming"
"Open YouTube"
"Exit"
```
